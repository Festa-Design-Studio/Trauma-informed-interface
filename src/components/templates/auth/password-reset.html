Password Reset Template Documentation

Usage

This template provides a trauma-informed password reset request page that normalizes forgetting passwords and creates a clear path to recovery.

Key Components

  • Supportive messaging around password recovery
  • Simple email input with clear instructions
  • Step-by-step explanation of the reset process
  • Easy navigation back to the login page
  • Alternative support option via direct contact

Trauma-Informed Considerations

  • Non-judgmental language that normalizes forgetting passwords
  • Clear, predictable process that reduces anxiety
  • Supportive messaging that avoids shame or blame
  • Multiple paths to get help (self-service and direct support)
  • Transparent next steps that create predictability
  • Simple design with minimal distractions
  • Emergency exit option for immediate departure
