Account Creation Template Documentation

Usage

This template provides a trauma-informed account creation process that collects necessary information while allowing users to set preferences that support their unique needs.

Key Components

  • Logically grouped form sections with clear headers
  • Learning preference options that allow for personalization
  • Content warning preferences to give users control
  • Password strength visual feedback
  • Alternative account creation methods
  • Clear consent checkboxes with accessible terms links

Trauma-Informed Considerations

  • Form divided into logical sections to reduce cognitive load
  • Learning preferences collection demonstrates commitment to personalization
  • Content warning preferences established at signup to prevent unexpected encounters
  • Clear, simple language throughout
  • Optional newsletter signup (not pre-checked)
  • Multiple options for account creation
  • Emergency exit button for quick navigation away if needed
