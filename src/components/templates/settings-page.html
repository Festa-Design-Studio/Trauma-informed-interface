Settings Page Template Documentation

Usage

This template provides a comprehensive user preferences interface that allows learners to customize their experience according to their needs, particularly focusing on trauma-informed considerations.

Key Components

  • Standard account management options (profile, password)
  • Accessibility controls for text size and motion
  • Content filtering preferences for sensitive topics
  • Content warning controls for customizing the learning experience
  • Topic-specific content warning preferences

Trauma-Informed Considerations

  • User empowerment through extensive customization options
  • Control over content detail levels to prevent overwhelm
  • Ability to opt in or out of specific types of content
  • Options to reduce motion and autoplay which can be triggering
  • Topic-specific content warning preferences respect individual triggers
  • Clear, non-judgmental explanation of each setting
  • Reinforcement that settings can be changed at any time
