Progress Dashboard Template Documentation

Usage

This template provides a comprehensive dashboard for users to track their learning progress across multiple courses. It serves as a central hub for learning management and progress visualization.

Key Components

  • Emergency exit button (consistent with other pages)
  • Overall progress visualization section
  • Course-specific progress trackers
  • Module-level completion status indicators
  • Supportive messaging that emphasizes self-paced learning
  • Course recommendations based on user interests/progress

Trauma-Informed Considerations

  • Non-pressuring language focused on progress rather than deficits
  • Supportive messaging that validates different learning paces
  • Clear, consistent visual progress indicators
  • Gentle color distinctions between completion states
  • Focus on achievement rather than incomplete work
  • Easy navigation to continue learning from where the user left off
  • Spacious layout that prevents cognitive overwhelm
