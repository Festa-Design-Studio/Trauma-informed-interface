Usage Guidelines

Usage:

  • Use list items to organize related information in scannable formats
  • Choose appropriate list style based on content importance and visual hierarchy
  • For feature lists with check marks, use when highlighting key benefits
  • For complex information requiring more description, use the benefits list with icons
  • Maintain consistent spacing within each list type

Accessibility:

  • Always use semantic <ul> and <li> elements for proper screen reader interpretation
  • Ensure sufficient color contrast for icons and text
  • Provide text alternatives for meaningful icons (via aria-label or visible text)
  • Maintain minimum text size of 14px (text-small) for readability
  • Use flexbox alignment to ensure proper spacing with assistive technologies

Trauma-informed Considerations:

  • Lists provide predictable structure that helps reduce cognitive load
  • Clear visual indicators (bullets, icons) support quick scanning without overwhelm
  • Soft color palette for indicators avoids triggering stress responses
  • Generous spacing creates visual breathing room between items
  • Consistent patterns across list types build user trust through familiarity
  • Inclusive, supportive language in list content fosters psychological safety
  • Different list styles accommodate varying information complexity needs
