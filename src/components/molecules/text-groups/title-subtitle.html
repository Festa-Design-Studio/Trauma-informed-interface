Usage Guidelines

Usage:

  • Use title-subtitle pairs to create clear information hierarchy
  • Match style to content importance (larger for primary content, smaller for supporting elements)
  • Maintain consistent spacing between title and subtitle
  • Use appropriate color pairings to indicate relationship between elements

Accessibility:

  • Maintain proper heading hierarchy for screen readers
  • Don't skip heading levels in the page structure
  • Ensure subtitles have sufficient contrast ratio
  • Use semantic HTML elements appropriately

Trauma-informed Considerations:

  • Consistent visual patterns create predictability
  • Clear hierarchy helps users quickly understand content importance
  • Subtitles provide additional context to reduce uncertainty
  • Supportive language in subtitles creates psychological safety
  • Generous spacing creates visual breathing room
  • Thoughtful color usage guides attention without overwhelming
  • Multiple variants accommodate different content needs and contexts
