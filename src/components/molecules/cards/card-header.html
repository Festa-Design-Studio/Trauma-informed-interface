Usage Guidelines

Usage:

  • For the top section of card components
  • To provide a clear title and context for the card content
  • As a container for metadata about the card content
  • In combination with card bodies and footers for complete cards

Variations:

  • Simple header with just a title
  • Header with subtitle for additional context
  • Header with icon for visual reinforcement
  • Header with status badge for indicating state
  • Header with content warning for trauma-informed signaling

Accessibility:

  • Use proper heading hierarchy (typically h3 within cards)
  • Ensure adequate color contrast (minimum 4.5:1 for text)
  • Include descriptive alt text for any icons
  • Content warnings should be at the top and clearly visible

Trauma-informed Considerations:

  • Clear headings establish predictability
  • Content warnings are prominently displayed at the top
  • Icons and visual elements are gentle and supportive
  • Adequate spacing reduces visual overload
  • Border separates the header from content for clarity
  • Consistent visual patterns throughout card headers
  • Metadata provides clear expectations about content
  • Neutral color scheme avoids triggering stress responses
  • Rounded corners soften the visual appearance
