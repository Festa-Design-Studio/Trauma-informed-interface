Usage Guidelines

Usage:

  • For presenting the main content of a card
  • Can contain text, images, lists, progress indicators, etc.
  • Should maintain consistent padding and spacing
  • Can include highlighted sections for important information

Variations:

  • Text-only for simple content
  • Image and text for visual reinforcement
  • Lists for structured content
  • Highlighted sections for emphasis
  • Progress indicators for tracking completion

Accessibility:

  • Ensure text has sufficient contrast (minimum 4.5:1)
  • Include descriptive alt text for images
  • Use semantic HTML elements for content structure
  • Maintain adequate spacing between elements

Trauma-informed Considerations:

  • Clear, readable text reduces cognitive load
  • Consistent padding creates visual stability
  • Gentle highlighting avoids jarring visual emphasis
  • Progress indicators provide orientation and predictability
  • Lists break down information into manageable chunks
  • Rounded corners create softer, less threatening appearance
  • Neutral color palette avoids triggering stress responses
  • Generous spacing improves readability and reduces visual stress
  • Images maintain consistent aspect ratio for visual predictability
