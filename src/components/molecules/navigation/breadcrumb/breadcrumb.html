Usage Guidelines

Usage:

  • Use breadcrumb navigation to show the user's current location in the site hierarchy
  • Implement on pages that are nested at least two levels deep in the site structure
  • Place breadcrumbs at the top of content area, beneath the main navigation

Accessibility:

  • Use proper semantic markup with <nav>, <ol> tags
  • Include aria-label="Breadcrumb" on the navigation element
  • Ensure sufficient color contrast for text links
  • Provide visible focus states for keyboard navigation

Trauma-informed considerations:

  • Breadcrumbs provide orientation and control, helping users feel grounded within complex interfaces
  • Clear path visualization reduces digital disorientation that can trigger anxiety
  • Consistent behavior across pages builds trust and predictability
  • Consider condensing breadcrumbs on mobile to prevent overwhelming users with limited screen space
  • Provide clear click targets with adequate spacing to reduce precision stress
