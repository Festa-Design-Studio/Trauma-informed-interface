Component Documentation

Usage

  • For providing encouragement and validation
  • To remind users about self-care during difficult content
  • To suggest grounding exercises or breaks
  • To highlight available support resources
  • For timed check-ins during extended engagement

Variations

  • Simple supportive message for general encouragement
  • Grounding exercise prompt for emotional regulation
  • Resource links for additional support
  • Timed check-in for encouraging breaks
  • Dismissible message for recurring visitors

Accessibility

  • Use adequate color contrast (minimum 4.5:1 for text)
  • Include descriptive headings and clear instructions
  • Make buttons large enough for easy interaction
  • Provide keyboard-accessible controls

Trauma-informed Considerations

  • Positive, non-judgmental language validates user experience
  • Gentle reminders about self-care promote agency and control
  • Calming green color scheme promotes sense of safety
  • Offers practical resources without overwhelming
  • Timed check-ins help prevent emotional flooding
  • Dismissible option respects user autonomy
  • Icons reinforce positive, supportive messaging
  • Rounded corners and generous padding create visual comfort
  • Multiple support options acknowledge diverse needs
