Body Text Typography Components

A collection of body text styles using the Montserrat typeface, designed for trauma-informed digital experiences with excellent readability.

Regular Body Text

This is standard body text, designed for optimal readability with comfortable line length and spacing. The Montserrat font provides excellent legibility while maintaining a warm, approachable feeling that helps create psychological safety.

<p class="font-montserrat font-regular text-body text-grey-800 mb-4 max-w-prose"> This is standard body text, designed for optimal readability with comfortable line length and spacing. </p>

Lead Paragraph Text

This is lead paragraph text, slightly larger and with increased line spacing. This style is ideal for introductory paragraphs or important information that requires extra emphasis without creating visual stress.

<p class="font-montserrat font-regular text-lead text-grey-900 mb-6 max-w-prose leading-relaxed"> This is lead paragraph text, slightly larger and with increased line spacing. </p>

Body text can include inline links that use a distinctive but non-jarring style. The underline creates clear affordance while the subtle hover effect provides feedback without being visually disruptive.

Usage Guidelines

Usage:

  • Regular body text for most content
  • Lead paragraph for introductions or important content
  • Include links, emphasis, and small text as needed
  • Always maintain maximum width for optimal readability

Accessibility:

  • Maintain sufficient font size (minimum 16px/1rem for body text)
  • Ensure adequate line spacing (1.5 times font size minimum)
  • Use adequate color contrast (minimum 4.5:1 for normal text)
  • Links should be distinguishable by more than color alone

Trauma-informed Considerations:

  • Clean, readable font reduces cognitive load
  • Generous line height improves readability and reduces visual stress
  • Maximum width constraint (max-w-prose) prevents overwhelming line lengths
  • Consistent visual patterns create predictability
  • Subtle but clear link styling indicates interactivity without disruption
  • Smooth transitions for interactive elements avoid jarring changes
  • Warm, neutral color palette avoids triggering stress responses
