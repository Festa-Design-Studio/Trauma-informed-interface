Usage Guidelines

Usage:

  • Create clear visual hierarchy with appropriate heading levels
  • Always use headings in sequential order (H1 → H2 → H3, etc.)
  • H1 for page titles, H2 for major sections, H3+ for subsections

Accessibility:

  • Proper heading hierarchy is essential for screen reader navigation
  • Don't skip heading levels (e.g., don't jump from H2 to H4)
  • Use semantic heading elements (h1-h6) rather than styled paragraphs
  • Maintain sufficient contrast ratios (minimum 4.5:1 for normal text)

Trauma-informed Considerations:

  • Font choices avoid overly decorative or difficult-to-read styles
  • Playfair Display for major headings provides elegant, balanced appearance
  • Montserrat for smaller headings maintains clean readability
  • Generous line height and letter spacing improve readability
  • Consistent color system creates predictable visual hierarchy
  • Progressive size reduction creates clear information structure
  • Adequate margin provides visual breathing room
  • Responsive sizing ensures readability across devices
