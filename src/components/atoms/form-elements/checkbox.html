Checkbox Component

Checkboxes allow users to select multiple options from a list or to toggle a single option on/off. They provide clear, binary choices and are essential for consent mechanisms in trauma-informed interfaces.

Standard Checkbox

Checkbox with Help Text

We'll only email you important updates, and you can unsubscribe anytime.

Content Warning Checkbox

The following module includes descriptions of traumatic experiences. Please confirm you wish to proceed.

Usage Guidelines

Usage:

  • For binary choices (yes/no, opt-in/opt-out)
  • For selecting multiple options from a list
  • For consent confirmations and acknowledgments

Accessibility:

  • Use proper label associations with 'for' and 'id' attributes
  • Maintain adequate size for touch targets (min 24x24px)
  • Provide descriptive text for what checking the box means
  • Use aria-describedby for additional descriptions

Trauma-informed considerations:

  • Clear, honest labeling about what the checkbox controls
  • No pre-checked boxes for consent-related items
  • Visual highlighting for content warnings or consent acknowledgments
  • Supportive, non-judgmental helper text
  • Content warnings given appropriate visual prominence
  • Custom styling maintains familiar checkbox appearance for recognition
  • Smooth transition states avoid jarring visual changes
