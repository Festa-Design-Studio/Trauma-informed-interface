Select Component

Dropdown menus for selecting from a list of options

Helper text with additional guidance

Please select an option to continue

Usage Guidelines

Usage:

  • For selecting one option from a list of many choices
  • When space is limited but many options are available
  • For categorization or filtering of content

Accessibility:

  • Use proper labels with 'for' and 'id' associations
  • Include descriptive placeholder text
  • Ensure keyboard navigability
  • Consider using the native select element for best screen reader support

Trauma-informed considerations:

  • Clear labeling helps users understand what they're selecting
  • Helper text provides additional context without overwhelming
  • Organized options reduce cognitive load
  • Predictable behavior builds trust
  • Visual feedback confirms user actions
  • Default options are thoughtfully chosen to be neutral
  • Error messages are supportive, not punitive
