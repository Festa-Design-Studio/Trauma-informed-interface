Emergency Exit Button Component

The emergency exit button provides an immediate, always-accessible escape route for users who feel triggered or overwhelmed. It is a key trauma-informed safety feature.

Default Emergency Exit Button

<button class="bg-rhodochrosite-rose-600 ... flex items-center gap-2"><svg ... />Exit to Safe Space</button>

Use for immediate, always-accessible escape routes for users who feel triggered or overwhelmed. The rose color signals urgency without using alarming red, supporting user safety and calm.

Full-width Emergency Exit Button

<button class="w-full bg-rhodochrosite-rose-600 ... flex items-center gap-2"><svg ... />Exit to Safe Space</button>

Use full-width for maximum visibility and accessibility, especially in mobile or modal contexts where a quick exit is critical for user safety.

Small Emergency Exit Button

<button class="bg-rhodochrosite-rose-600 ... py-xs px-sm ... text-small flex items-center gap-2"><svg ... />Exit</button>

Use for compact layouts or sticky/fixed positions where space is limited but immediate access is still required.

Disabled Emergency Exit Button

<button class="bg-rhodochrosite-rose-200 ... opacity-60 cursor-not-allowed flex items-center gap-2" disabled><svg ... />Exit to Safe Space</button>

Use to indicate the emergency exit is temporarily unavailable. The faded color and cursor communicate that the action is not possible, but the button remains visible for reassurance.

Loading State Emergency Exit Button

<button class="... flex items-center ..." disabled><svg class="animate-spin ..." ... />Exiting...</button>

Use when the emergency exit is processing. The spinner and disabled state provide clear, non-alarming feedback that the system is working.

Buttons on Colored Backgrounds & Dark Mode

On Colored Backgrounds

<button class="bg-rhodochrosite-rose-DEFAULT text-selenite-white-50 ...">Emergency Exit</button>

On colored backgrounds, use a strong, high-contrast color (e.g., rose) for the emergency exit button to ensure it stands out and is always visible.

In Dark Mode

<button class="bg-rhodochrosite-rose-600 text-selenite-white-50 ... dark:bg-rhodochrosite-rose-400 dark:text-grey-900">Emergency Exit</button>

For dark mode, use a saturated rose color and ensure the text remains readable. Use dark: classes for Tailwind or custom CSS for other frameworks.

Accessibility

  • Emergency exit buttons must always be visible and accessible.
  • Use high-contrast colors and large touch targets.
  • Test on all backgrounds and in dark mode for visibility.
