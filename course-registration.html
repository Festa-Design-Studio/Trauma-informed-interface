Course Registration Template Documentation

Usage

This template provides a comprehensive, trauma-informed course registration form that collects necessary information while respecting user agency and comfort.

Key Components

  • Clear progress indicator showing registration steps
  • Course summary with transparent pricing information
  • Options for learning accommodations
  • Content sensitivity preferences
  • Detailed "What's Included" section to set clear expectations
  • Transparent consent and terms section
  • Emergency exit button for quick navigation away

Trauma-Informed Considerations

  • Clear content warning preferences that give users control over information presentation
  • Options to specify topics to avoid without requiring detailed explanations
  • Comprehensive accommodation options that normalize various needs
  • Progress indicators to create predictability
  • Clear explanations of what's included to reduce uncertainty
  • Clearly labeled required vs. optional fields
  • Thoughtful form organization with related fields grouped together
  • Ample whitespace and clean layout to reduce cognitive load
