Blog Grid Template Documentation

Usage

This template presents a comprehensive blog and resources hub that allows users to browse, search, and filter content in a trauma-informed manner. It's designed to provide easy access to various content types while maintaining clear content warnings where needed.

Key Components

  • Emergency exit button for quick navigation away from any potentially triggering content
  • Content filters including explicit content warning filter options
  • Featured article section with prominent placement
  • Blog card components with optional content warnings
  • Case study and resource components for different content types
  • Clear metadata showing authors and publication dates
  • Accessible pagination system for navigating through content

Trauma-Informed Considerations

  • Content warning overlays for potentially triggering articles
  • Dedicated content warning filter to allow users to control exposure
  • Clear categorization through badges and tags helps users make informed choices
  • Gentle image hover animations provide subtle but noticeable feedback
  • Consistent card structures create predictability across content types
  • Author information creates a sense of human connection and accountability
  • Readable excerpts that provide enough context without overwhelming details
